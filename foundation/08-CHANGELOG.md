# BrytThrive Foundation Changelog

> **Purpose:** Permanent record of all amendments to the BrytThrive Foundation documents.
> **Authority:** Changes to ratified constitutional principles require a formal amendment recorded here.

---

## Format

Each entry records:

- **Version** — Semantic version of the Constitution affected (or `N/A` for non-constitutional changes)
- **Amendment / Change** — Name and description
- **Date** — Date ratified or merged
- **Documents affected** — Which Foundation files changed
- **Status** — Ratified / Pending / Superseded

## Future Amendment Process

To propose a Constitutional Amendment:

1. Draft the proposed article or change in a pull request against `foundation/01-CONSTITUTION.md`.
2. Reference which existing article(s) the amendment extends, modifies, or supersedes.
3. Add a Pending entry to this Changelog.
4. Obtain agreement from product leadership.
5. Merge and update the entry status to Ratified.
6. Update the Constitution version number.

Non-constitutional changes to other Foundation documents (e.g., Glossary additions, Product Bible updates) do not require the full amendment process — but must still be recorded here.
